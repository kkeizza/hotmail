# Hotmail Validator by Country

A powerful tool to validate Hotmail/Outlook accounts and sort them by country.

## Features

- ✅ Validate Hotmail/Outlook accounts
- 🌍 Sort results by country
- 📊 Live statistics display
- 🔄 Multi-threaded processing
- 📱 Telegram integration
- 💾 Automatic saving to country-specific files
- 🚀 High speed with rate limiting

## Installation

### Windows
```bash
pip install -r requirements.txt
python hotmail_validator.py
